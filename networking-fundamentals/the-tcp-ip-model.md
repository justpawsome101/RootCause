# The TCP/IP Model

The TCP/IP model (also called the Internet model) predates OSI and is what the internet actually runs on. It SQAUSHES the seven OSI layers into four:

| TCP/IP Layer          | Corresponds to OSI |
| --------------------- | ------------------ |
| Application           | Layers 5, 6, 7     |
| Transport             | Layer 4            |
| Internet              | Layer 3            |
| Network Access (Link) | Layers 1, 2        |

In practice, people use the two models interchangeably when talking about specific protocols. When someone says "Layer 3 attack," they mean the network/IP layer in both models. Don't get tripped up by the terminology.....understand the concepts and the models will take care of themselves.
